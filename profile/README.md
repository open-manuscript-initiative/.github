## Our Mission

The Open Manuscript Initiative (OMI) is an open-source project developing semantic, platform-independent standards for scholarly authoring, peer review, publishing, and long-term preservation.

Instead of formatting documents for individual publishers, OMI enables researchers to create a single semantic manuscript that can be published everywhere.

---

## Core Principles

- Semantic authoring
- Open standards
- Platform independence
- Interoperability
- Accessibility
- Long-term preservation
